# Adders Presentation

A cross-platform Flutter application that serves as a presentation covering the concepts of basic digital logic adders.

## Features
- **Presentation Mode**: Slide-based interface with navigation.
- **Topics Covered**:
  - Binary Addition Basics
  - Half Adder (Logic, Truth Table, Circuit)
  - Full Adder (Logic, Truth Table, Circuit)
  - Ripple Carry Adder (Chaining Full Adders)
- **Responsive Layout**: Adapts to mobile and desktop screens.

## Setup & Run
1. Ensure Flutter is installed.
2. Run `flutter pub get` to install dependencies.
3. Run `flutter run` to launch the app.
